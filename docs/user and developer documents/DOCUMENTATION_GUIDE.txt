HOW TO ADD SCREENSHOTS
================================================================================

STEP 1: Identify Screenshot Locations
- Read each [SCREENSHOT X] placeholder in the guide
- Understand what it should show from the description

STEP 2: Capture Screenshots
- Login to your actual SPECTROPY system
- Navigate to each screen mentioned
- Take clean screenshots (1920x1080 recommended)
- Crop unnecessary borders
- Ensure text is readable
- Use consistent styling/theme

STEP 3: Save Screenshots
- Save as: [GUIDE_NAME]_screenshot_[NUMBER].png
  Example: student_guide_screenshot_1.png
- Create a folder: /screenshots/
- Organize by guide: /screenshots/student/, /screenshots/teacher/, /screenshots/admin/

STEP 4: Insert into HTML
- Open HTML file in text editor
- Find [SCREENSHOT X] placeholder
- Replace with: <img src="path/to/screenshot.png" alt="Description" style="max-width: 100%; margin: 20px 0; border: 1px solid #ddd;">
- Example:
  <div class="screenshot-placeholder">
      <img src="../screenshots/student/student_guide_screenshot_1.png" alt="Login Page" style="max-width: 100%; margin: 20px 0; border: 1px solid #ddd;">
      <span class="screenshot-label">Login page with email and password fields</span>
  </div>

STEP 5: Test
- Open HTML file in browser
- Verify image displays correctly
- Check on mobile view
- Test print to PDF

================================================================================
CONVERTING TO PDF
================================================================================

METHOD 1: Chrome/Edge Browser (Recommended)
1. Open HTML file in Chrome or Edge browser
2. Press Ctrl+P (or Cmd+P on Mac)
3. Click "Save as PDF"
4. Choose location and filename
5. Adjust settings:
   - Paper size: A4 or Letter
   - Margins: Normal or Custom
   - Scale: 100%
   - Background graphics: ON (for colored sections)
6. Click "Save"

METHOD 2: Firefox Browser
1. Open HTML file in Firefox
2. Press Ctrl+P (or Cmd+P on Mac)
3. Select "Print to File"
4. Choose PDF format
5. Click "Print"

METHOD 3: Online Converter
1. Visit: https://cloudconvert.com/html-to-pdf
2. Upload HTML file
3. Convert to PDF
4. Download result

METHOD 4: Professional Approach (Pandoc)
1. Install Pandoc: https://pandoc.org/installing.html
2. Run: pandoc -f html -t pdf input.html -o output.pdf
3. Result: High-quality PDF with table of contents

================================================================================
CONVERSION TO DOCX (Word Format)
================================================================================

METHOD 1: Online Converter
- Visit: https://cloudconvert.com/html-to-docx
- Upload HTML file
- Select DOCX format
- Download converted document

METHOD 2: LibreOffice (Free)
1. Open HTML file in LibreOffice Writer
2. File → Export as PDF or Save As → Microsoft Word (.docx)

METHOD 3: Pandoc
- Run: pandoc -f html -t docx input.html -o output.docx
